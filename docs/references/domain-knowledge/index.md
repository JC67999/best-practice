## How to Use

### Automatic (Recommended)
Just mention unfamiliar domain concepts in your requests:
```
"Optimize the transformer attention mechanism"
→ Claude researches if needed, stores knowledge, then helps
```

### Manual
Explicitly request research:
```
"Research [topic] for this project"
→ Claude fetches information and stores in knowledge base
```
